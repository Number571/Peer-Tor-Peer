# PeerTorPeer

#### Compile, Install and Run:
```
$ make
$ make install
$ make run
```

#### Commands in the P2rP client
```
| :exit = Exit from P2rP
| :config = Reload config
| :connect = Connect to client(s) P2rP
| :disconnect = Disconnect from client(s) P2rP
| :hostname	= Print hostname
| :network = Print connections
```
